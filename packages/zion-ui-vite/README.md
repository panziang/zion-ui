- 🐆 全面基于 Vite，速度够快
- 🤟 极致的开发体验

- ✂️ 完善的代码提示
- 🪐 不同打包模式，兼容不同项目


## 安装

```bash
npm i zion-ui-vite
```

## 快速开始

```js
import Vue from 'vue'
import ZionUI from 'zion-ui-vite'
import "zion-ui-vite/style.css";

createApp(App)
    .use(ZionUI)
    .mount("#app");
```
